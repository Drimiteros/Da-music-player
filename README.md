# Da Music Player 🎵  

A simple music player built in C++ that allows you to:  
✅ Search directories for music files  
✅ Play and stop music  
✅ View timestamps  

Simple stuff!  

---

## 🛠️ Tech Stack  
- **C++ 17 and above**  
- **SFML 2.6.0**
- **Visual Studio 2022**
